# C/C++ Dev Environment (Make)

## devenv.nix

```nix
{ pkgs, ... }:
let
  clang = pkgs.clang_19;
  clang-tools = pkgs.clang-tools_19;
in
{
  languages.c = {
    enable = true;
    lsp.package = clang-tools;
  };

  packages = [
    clang
    clang-tools
    pkgs.bear        # generates compile_commands.json from make
    pkgs.pkg-config
    pkgs.gdb
  ];

  env.CC = "${clang}/bin/clang";
  env.CXX = "${clang}/bin/clang++";
}
```

## compile_commands.json — the key to std:: navigation

clangd needs this to know where headers live (including Nix store paths).
Generate it with `bear` by wrapping your make invocation:

```bash
bear -- make
```

This produces `compile_commands.json` at the project root. Rerun after
adding/removing source files.

## .clangd

`Add` appends flags to every file clangd parses — use it to set the standard
when you don't want to bake it into the Makefile's `CXXFLAGS`.

```yaml
CompileFlags:
  Add: [-std=c++20]
  Compiler: clang++
```

If `compile_commands.json` is present, `Add` flags are merged with the real
compile flags from the file.

## .clang-format

```yaml
BasedOnStyle: LLVM

ColumnLimit: 100
```

## Neovim — lsp.lua

```lua
clangd = {
  cmd = {
    "clangd",
    "--background-index",
    "--clang-tidy",
  },
  capabilities = {
    offsetEncoding = "utf-8",
  },
  settings = {
    fallbackFlags = { "-std=c++20" },  -- safety net if compile_commands.json is absent
  },
},
```

## Neovim — conform.lua

```lua
formatters_by_ft = {
  c = { "clang-format" },
  cpp = { "clang-format" },
},
formatters = {
  clang_format = {
    prepend_args = { "--style=LLVM" },  -- overridden by .clang-format if present
  },
},
```

## Common commands

```bash
bear -- make               # build + generate compile_commands.json
make test                  # run tests
make clean                 # clean build artifacts
clang-tidy src/main.cpp    # lint
clang-format -i src/**/*.cpp
```
